# oshilog マーケティング戦略

> 推し活の記録を、もっと楽しく。

---

## 1. 状況分析（現状把握）

### アプリ概要

| 項目 | 内容 |
|------|------|
| アプリ名 | oshilog |
| カテゴリ | 家計管理 / ライフスタイル |
| 対象OS | iOS / Android |
| 価格モデル | フリーミアム（無料 + ¥480/月プレミアム） |
| コア機能 | 推し活支出の記録・管理・予算管理 |

### 強み（Strengths）

- **ニッチ特化**: 汎用家計簿アプリにない「推し」単位での管理
- **UX設計**: 推し活ユーザーの行動に合わせたカテゴリ設定（グッズ/チケット/配信/写真集/その他）
- **デザイン**: ターゲット層（若い女性ファン）に刺さるピンク/パープルのグラデーション
- **低価格**: ¥480/月という手頃なプレミアム料金

### 弱み（Weaknesses）

- 認知度ゼロからのスタート
- Google OAuth 未実装（ログイン障壁）
- 広告収益モデル未整備

### 機会（Opportunities）

- 推し活市場の急成長（年間消費額1兆円超と推計）
- SNS拡散文化との相性の良さ
- 「推し活」概念の海外（韓国・台湾・東南アジア）への広がり

### 脅威（Threats）

- 汎用家計簿アプリ（Zaim, マネーフォワード等）が類似機能追加の可能性
- ユーザーのスプレッドシート自作による代替

**「推し活を始めた10代学生」**

- 予算が限られるため、管理ニーズが高い
- バイト代の推し活配分を計画したい
- SNSで「推し活記録」を共有したい

## 3. ポジショニング

### コアバリュープロポジション

> **「推しへの愛を、数字で証明する」**

oshilogは単なる家計簿アプリではなく、**推し活ライフスタイルの記録ツール**として位置づける。

```
                    推し特化 ←──────────────→ 汎用
                       │                        │
                       │         oshilog ●      │
   感情的価値高い ───────┼────────────────────────┼─── 機能的価値高い
                       │                        │
                       │  マネーフォワード ●     │
                       │    Zaim ●              │
```

### メッセージフレームワーク

- **課題**: 「推し活費がいつの間にか増えていて、月末に焦る」
- **解決**: 「推しごとに記録すれば、愛の重さが見える」
- **感情**: 「使った金額が、推しへの本気の証明になる」

## 4. マーケティング戦略

### フェーズ1: 認知獲得（リリース〜3ヶ月）

#### 4.1 SNSオーガニック戦略

**X（旧Twitter）**

- アカウント：`@oshilog_app`
- 投稿頻度：1日1〜2回
- コンテンツ例：
  - 「今月の推し活費、公開します📊 #oshilog #推し活記録」
  - 推し活あるあるミーム
  - ユーザーの記録スクリーンショット（リポスト）
  - 機能紹介の短尺動画

**ハッシュタグ戦略**

| 種別 | タグ例 |
|------|--------|
| ブランドタグ | `#oshilog` `#推し活ログ` |
| 拡散タグ | `#推し活` `#推し活費公開` `#推し貯金` |
| ジャンルタグ | `#Kpop沼` `#2_5次元` `#Vtuber活` |

**TikTok / Instagram Reels**

- フォーマット：15〜30秒のアプリ操作動画
  - 「推し活費の入力がこんなに簡単」
  - 「1ヶ月の推し活費を振り返ってみた」
  - 「推し別グラフが可愛すぎる」

#### 4.2 インフルエンサーマーケティング

| 層 | フォロワー数 | アプローチ |

| ミドルインフルエンサー | 50,000〜200,000 | 有償案件（¥30,000〜¥100,000） |

**ターゲットジャンル**

- K-POP / 2.5次元系インフルエンサー
- 家計管理 × オタク系アカウント

**アプローチ文テンプレート**
```
推し活費の管理アプリ「oshilog」を開発しました。
推しごとに支出を記録でき、予算管理もできます。
ぜひご紹介いただけますでしょうか？
```

#### 4.3 コミュニティマーケティング

- **X推し活クラスタへの積極参加**: 推し活費公開ツイートにリプライ

- **note記事**: 「推し活費を管理するメリット」「推し活と家計管理の両立術」

### フェーズ3: 収益化・グロース（6ヶ月〜）

#### 4.6 プレミアム転換戦略

**フリーミアム転換率目標: 5〜8%**

| 施策 | 内容 |
|------|------|
| 無料→有料の壁を設計 | 3推し制限を自然に体験させる |
| 期間限定トライアル | 14日間プレミアム無料体験 |
| 年払いオプション | ¥4,800/年（2ヶ月分お得）導入 |
| 特定タイミングでのアップグレード促進 | 推し3人目追加時にアップグレード訴求 |

#### 4.7 App Store 最適化（ASO）

**タイトル**: oshilog - 推し活費管理アプリ

**キーワード（App Store）**:
```
推し活,家計管理,推し,支出記録,オタク,Kpop,アイドル,推し費,家計簿,推し活費
```

**スクリーンショット設計**:
1. ホーム画面：「今月の推し活費が一目でわかる」
2. 支出入力：「カテゴリ別に簡単入力」
3. 推し一覧：「推しごとにカラーで管理」
4. 予算管理：「月予算を設定してオーバーを防ぐ」
5. 統計グラフ：「推し活ライフを数値で振り返る」

**説明文冒頭（重要）**:
```
推し活費を、もっと賢く。もっと楽しく。

「今月いくら使ったっけ？」そんな悩みを解決します。
oshilogは推し活専用の支出管理アプリです。

✨ 特徴
• 推しごとに支出を分類・管理
• グッズ/チケット/配信など推し活専用カテゴリ
• 月予算設定でオーバー防止
• シンプルで可愛いデザイン
```
